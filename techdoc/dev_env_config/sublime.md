

# 配置更智能可用的文本编辑器 sublime

常用命令

```
CTRL + Alt + P 调出命令端
opt + cmd + f 全局替换
```


推荐插件
> 插件搜索：[https://packagecontrol.io/search/](https://packagecontrol.io/search/)

- markdown 编辑插件： `Markdown Editing`
- markdown 预览插件： `Marddown Preview`
- 文本格式化插件： `HTML-CSS-JS Prettify` (依赖 nodejs )
- git 插件： `git`
